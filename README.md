# superstore-sales-analysis


## Project Overview
This project analyzes the Superstore dataset using
Python and Power BI.

## Key Findings
- Technology is the most profitable category.
- Tables and Bookcases generate losses.
- Higher discount rates are associated with lower profitability.
- Sales exhibit a seasonal pattern over time.

## Tools Used
- Python (Pandas)
- Power BI
